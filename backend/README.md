# Mini CRM Backend

This is the backend for the Mini CRM platform (Node.js, Express, MongoDB).

## Initial Structure

- `index.js`: Entry point
- `package.json`: Project metadata
- `/routes`: API routes
- `/models`: Mongoose models
- `/controllers`: Business logic
- `/config`: DB and other configs
- `/docs`: Swagger API docs

## Next Steps

- Initialize Node.js project
- Install dependencies (express, mongoose, swagger-ui-express, etc.)
- Set up basic Express server
- Connect to MongoDB
- Add Swagger UI
